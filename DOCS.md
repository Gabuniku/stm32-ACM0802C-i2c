# リファレンス

LCDの操作はすべて`acm0802c_t`構造体を介して行います．

## 構造体,列挙等

### 構造体

`acm0802c_t`構造体は5個のフィールドを持ちますが，実際に扱うのは2つだけです．
以下の表はに2つのフィールドについての表です．
| 名前    | 型                 | 意味                                      |
| :------ | :----------------- | :---------------------------------------- |
| hi2c    | I2C_HandleTypeDef* | i2cペリフェラルのハンドルのポインタです． |
| address | uint8_t            | LCDのアドレスです．                       |

### 列挙型

`DISPLAY_CONF`はLCDのディスプレイの設定を行う際に使うビットを列挙したものです．
以下の表はに`DISPLAY_CONF`についての表です．

| 名前        | 意味                       |
| :---------- | :------------------------- |
| BLINK_OFF   | カーソル位置が点滅しません |
| BLINK_ON    | カーソル位置が点滅します   |
| CURSOR_OFF  | カーソルを表示しません     |
| CURSOR_ON   | カーソルを表示します       |
| DISPLAY_OFF | 画面の表示をオフにします   |
| DISPLAY_ON  | 画面の表示をオンにします   |

`SHIFT_CONF`はLCDのシフトの設定を行う際に使うビットを列挙したものです．
以下の表はに`SHIFT_CONF`についての表です．

| 名前          | 意味                         |
| :------------ | :--------------------------- |
| SHIFT_DISPLAY | ディスプレイをシフトさせます |
| SHIFT_CURSOR  | カーソルをシフトさせます     |
| SHIFT_RIGHT   | 右にシフトします             |
| SHIFT_LEFT    | 左にシフトします             |


## 関数

### 設定

`void acm0802c_Begin(acm0802c_t *lcd)`

渡した`acm0802c_t`構造体のポインタに対して初期化を行います．
この関数で初期化される設定値はディスプレイ:オン，シフト:カーソル，右シフトです.

`void acm0802c_CursorOn(acm0802c_t *lcd)`

カーソルを表示します．

`void acm0802c_CursorOff(acm0802c_t *lcd)`

カーソルを消します．

`void acm0802c_BlinkOn(acm0802c_t *lcd)`

カーソル位置を点滅させます．

`void acm0802c_BlinkOn(acm0802c_t *lcd)`

カーソル位置の点滅を止めます．

### 表示
`void acm0802c_Render(acm0802c_t *lcd, const char *text)`
textに渡したchar型の配列を現在のカーソル位置に表示します．
**textの最後は必ずNULL文字=`'\0'`を入れてください．**
表示できる文字とその値はデータシート参照

### その他

`void acm0802c_Home(acm0802c_t *lcd)`

カーソル位置をホーム（左上）に戻します．

`void acm0802c_Clear(acm0802c_t *lcd)`

表示している文字を消します．
